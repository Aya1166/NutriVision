# NutriVision 🍽️

NutriVision is a modern nutrition and recipe analytics platform developed using Blazor and Entity Framework Core.

## 🚀 Features

- Ingredient management
- Recipe management
- Nutritional analysis
- Meal planner
- Calories calculation
- Smart nutrition insights
- Analytics dashboard
- Dynamic filtering & search
- Full CRUD operations

## 🛠️ Technologies Used

- Blazor Web App
- ASP.NET Core
- Entity Framework Core
- SQL Server
- LINQ
- Bootstrap / Custom CSS

## 📊 Analytics Features

- Total recipe calories
- Calories per serving
- Recipe distribution by category
- Cuisine analysis
- Daily meal calorie tracking

## ✨ Creativity Features

- Smart nutrition insights
- Interactive meal planner
- Modern UI/UX
- Dynamic dashboard
